== Description ==

= woo extra tabs =

It's a premium plugin but available for free.

This plugin extends WooCommerce to allow shop owners to add custom tabs to products. The tabs are displayed on the individual product pages to the right of the default "Description" tab.

Tab content areas use the standard WordPress text editor and may contain text, images, html or shortcodes.


== Frequently Asked Questions ==

= 1. Where do I go to add tabs to a product? =
You will see WOO EXTRA TAB plugin at dashboard menu. Go there and add menu.
Select woocommerce category where you want to show tabs or edit individual product and select the tab title.

= 2. Can I change tab title =
Yes you can, and it will be reflected to every product the tab is assigned.
